lwip_mmwlan_init_args Struct Reference
LwIP wrapper for Morse Micro Wireless LAN

Detailed Description

Initialise arguments structure.

This should be initialised using LWIP_MMWLAN_INIT_ARGS_DEFAULT. For example:

struct lwip_mmwlan_init_args args = LWIP_MMWLAN_INIT_ARGS_DEFAULT;
LWIP_MMWLAN_INIT_ARGS_DEFAULT
#define LWIP_MMWLAN_INIT_ARGS_DEFAULT
