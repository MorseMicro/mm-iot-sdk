lwIP embedded HTTP web server

The web page you are watching was served by a simple web server running on top of the lightweight TCP/IP stack lwIP, on a Morse Micro IoT SDK device.

This webpage is embedded in the binary running on the Morse Micro IoT SDK device, and is being delivered over Wi-Fi HaLow
